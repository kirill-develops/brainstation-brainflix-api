BrainFlix API Documentation

Endpoints

GET /videos

Returns an array of video objects

Example response:

[
    {   
        "id": "1af0jruup5gu", 
        "title": "BMX Rampage: 2018 Highlights", 
        "channel": "Red Cow", 
        "image": "https://i.imgur.com/l2Xfgpl.jpg" 
    },
    { 
        "id": "1ainjruutd1j", 
        "title": "Become A Travel Pro In One Easy Lesson", 
        "channel": "Todd Welch", 
        "image": "https://i.imgur.com/5qyCZrD.jpg"
    }
]

POST /videos

Creates a new video post and returns a detailed object of new video

Body must include all values

Post body response:

{
  "title": "How Bacon can save the World",
  "channel": "Ron Burgendy",
  "image": "https://i.imgur.com/l2Xfgpl.jpg",
  "video": "https://project-2-api.herokuapp.com/stream",
  "description": "It's salty, smoky, and sweet. It can make almost any dish better. It's bacon, and it's the best meat ever!"
}

PUT /videos/:videoId/likes

:videoID must be swapped out with the numeric id of a video as found in the list of videos

Toggle like counter up or down depending on the state on the client side

Returns oppisite LIKED state and a detailed object of LIKED video

POST /videos/:videoId/comments

:videoID must be swapped out with the numeric id of a video as found in the list of videos

Creates a new comment for a specific video

Post body response:

{
	"name": "Nigel",
	"comment": "This is a test"
}

DELETE /videos/:videoId/comments/:commentId

:videoID must be swapped out with the numeric ID of a video as found in the list of videos

:commentID must be swapped out with the numeric id of a comment as found in the list of comments for the given video

Returns a detailed object of a video with deleted comment
